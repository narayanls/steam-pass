## Features

- Simple GTK 4 interface for quickly switching between Steam accounts on Linux.
- Lightweight, with minimal dependencies.
- Built in Python, easy to hack and extend.

## Installation

### Arch Linux and derivatives

1. Download the package file: `steam-pass-r22.42f2bbf-1-any.pkg.tar.zst`.
2. Open a terminal in the folder where you downloaded the file.
3. Install the package with:

```bash
sudo pacman -U steam-pass-r22.42f2bbf-1-any.pkg.tar.zst
```

### Other Linux distributions (AppImage)

1. Download the AppImage file from the releases page.
2. Right click the file, open **Properties** and enable **Allow executing file as program** (or similar).
3. Double-click the AppImage.
4. When prompted, accept integrating the AppImage with your system (optional but recommended).

## Usage

After installing:

- Launch **steam-pass** from your application menu.

## Requirements

- Steam installed on your system and launched at least once.
